# Spark-Assignment-Week-6
Apache Spark Data Processing Pipeline

This project demonstrates the use of Apache Spark (PySpark) for efficient data processing and analysis. It includes reading CSV data with schema inference, performing DataFrame transformations such as filtering, selecting, renaming columns, casting data types, handling null values, removing duplicates, sorting, grouping, and adding new columns. A complete ETL pipeline was implemented to process the dataset and save the results in both CSV and Parquet formats. The project also explores key Spark concepts including Spark Architecture, Lazy Evaluation, DAG (Lineage Graph), Transformations vs. Actions, Shuffle, Predicate Pushdown, and performance optimization techniques for large-scale data processing.
